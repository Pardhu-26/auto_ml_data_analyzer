# 🤖 AutoML Data Analyzer Dashboard

An intelligent AutoML-powered data analysis dashboard that automates data preprocessing, visualization, machine learning model training, and insight generation with minimal user input. The application is built using Python, Scikit-learn, Pandas, and Streamlit to simplify data analysis and predictive modeling workflows.

---

# 📌 Project Overview

Data analysis and machine learning often require multiple manual steps including preprocessing, visualization, feature engineering, and model selection. This project automates these tasks through an interactive dashboard that enables users to upload datasets and generate insights, visualizations, and predictive models automatically.

The project demonstrates the complete end-to-end Machine Learning workflow including:

- Automated data preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Feature engineering
- Automated ML model training
- Model evaluation and comparison
- Interactive dashboard deployment

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/automl-data-analyzer.git
```

## 2️⃣ Navigate to Project Folder

```bash
cd automl-data-analyzer
```

## 3️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

## 4️⃣ Run the Streamlit Application

```bash
streamlit run app.py
```
